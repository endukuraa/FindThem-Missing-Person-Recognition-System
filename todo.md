# Missing Persons Dashboard - MVP Implementation

## Overview
Create an impressive modern website for a Missing Persons Dashboard with face recognition capabilities.

## Core Features to Implement
1. **Home Page** - Modern landing page with hero section and feature highlights
2. **Registration Page** - Form to register missing persons with photo upload
3. **All Cases Page** - Grid display of all registered cases with search/filter
4. **Match Page** - Face matching interface with drag-and-drop photo upload
5. **Help Page** - Information and guidance for users

## Files to Create/Modify
1. `src/pages/Index.tsx` - Modern home page with hero section
2. `src/pages/Registration.tsx` - Registration form with file upload
3. `src/pages/AllCases.tsx` - Cases display with grid layout
4. `src/pages/Match.tsx` - Face matching interface
5. `src/pages/Help.tsx` - Help and information page
6. `src/components/ui/file-upload.tsx` - Custom file upload component
7. `src/lib/api.ts` - API integration functions
8. `src/App.tsx` - Update routing structure
9. `index.html` - Update title and meta tags

## Design Principles
- Modern glassmorphism design with dark theme
- Smooth animations and transitions
- Responsive mobile-first design
- Accessible UI components
- Professional color scheme with accent colors
- Interactive elements with hover effects

## Tech Stack
- React with TypeScript
- Shadcn/UI components
- Tailwind CSS for styling
- React Router for navigation
- Framer Motion for animations (if needed)
